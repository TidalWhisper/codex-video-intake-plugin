## 目的

这份规则用来解决两个真实问题：

1. prompt 太松时，相邻镜头容易抽成同一个构图
2. 背影视角类 prompt 太空时，容易抽出黑边或假电影边框

目标不是把 prompt 写得更长，而是写得更收敛、更有镜头控制力。

## 总原则

一条 `Next Scene` 只服务一个镜头帧。

它必须同时交代：

1. 主角身份锁定
2. 这一个镜头里发生什么
3. 镜头长什么样
4. 与前后镜头相比，差异点在哪里

## 推荐结构

推荐固定顺序：

`人物锚点 + 当前动作 + 视角/朝向 + 景别 + 人物占比 + 环境信息 + 光线 + 情绪 + 约束语`

## 必写字段

每条 `Next Scene` 至少写清：

1. `同一位主角`
2. `同一张脸 / 同一发型 / 同一条裙子的版型轮廓`
3. `动作`
4. `视角`
5. `景别`
6. `人物在画面中的占比`
7. `光线`
8. `情绪推进`

## 镜头字段要写到什么程度

### 视角

不要只写：

- 正面
- 侧面
- 背影

要写成：

- 正面偏右三分之三视角
- 左侧 profile
- 从人物后方偏左的三分之四背影视角

### 景别

不要只写：

- 远景
- 中景

要写成：

- 极宽建立镜头
- 胸像中近景
- 膝上中景
- 全身长镜头

### 人物占比

不要省略。

要明确：

- 人物只占画面高度约三分之一
- 人物占画面约六成
- 人物保持全身入镜并留足地平线环境

## 相邻镜头差异规则

相邻镜头至少拉开两项差异：

1. 景别不同
2. 朝向不同
3. 人物占比不同
4. 动作状态不同
5. 机位位置不同

如果只换一句情绪词，不换镜头结构，极容易抽成重复画面。

## 背影视角专门规则

背影视角是高风险项，必须补充这些约束：

1. 明确是 `rear three-quarter` 还是 `full back view`
2. 明确画面必须 `full-frame`
3. 明确 `no black bars`
4. 明确 `no embedded border`
5. 明确保留发型长度、裙型轮廓和身体线条可读

## 禁止写法

禁止：

1. 一条里塞多行 `Next Scene`
2. 用抽象词替代镜头控制
   - “更有故事感”
   - “电影感更强”
   - “氛围更美”
3. 只写情绪，不写景别和占比
4. 只写“远景背影”，不写 full-frame 和边框限制
5. 同时要求两个互相冲突的镜头
   - 既要近景脸部细节，又要人物很小

## 第二轮 smoke 推荐分工

### S001

- 任务：建立环境
- 推荐：极宽建立镜头
- 人物：全身，小比例

### S002

- 任务：推进停顿
- 推荐：胸像中近景 / 左侧 profile
- 人物：大比例

### S003

- 任务：情绪松开
- 推荐：膝上中景 / 三分之三正面
- 人物：中高比例

### S004

- 任务：收束离开
- 推荐：后方偏左三分之四长镜头
- 人物：全身
- 额外：必须明确 `full-frame, no black bars, no embedded frame`

## 推荐句式模板

```text
Next Scene：同一位20岁出头的亚洲年轻女性，保持同一张脸、同一发型、同一条浅色长裙的版型、领口、腰线和整体轮廓不变。她正在____。镜头采用____视角与____景别，人物在画面中约占____，____必须清楚可读。黄昏海滩、晚霞、海风和海面反光继续保留，但不要盖过主角。暖金色夕阳____光线，情绪推进到____。full-frame image, no black bars, no embedded border, no extra people, realistic cinematic short film.
```

## 对代理的硬规则

1. 先收敛镜头语义，再考虑多抽
2. 发现相邻镜头过近时，优先改 `视角 + 景别 + 人物占比`
3. 发现背影视角出黑边时，优先补 `full-frame / no black bars / no embedded border`
4. 不要把 Stage 04 的长 prompt 直接整段塞进 `Next Scene`
