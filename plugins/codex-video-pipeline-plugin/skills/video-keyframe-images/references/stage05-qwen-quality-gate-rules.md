## 目的

这份规则只解决一个问题：

- `QwenEdit+NextScene` 即使真实出图成功，也不能默认等于可交付。

Stage 05 对这条 Qwen 参考图路线的判断标准不是：

- 工作流有没有跑完
- 图片有没有生成出来

而是：

- 这张图能不能作为后续 Stage 06 的可靠首尾关键帧

## 适用范围

只适用于：

- `stage05_realistic_cinematic_qwen_edit_nextscene_local`
- `reference_guided`
- 单主角
- 单参考图
- 单镜头执行模式

## 质检分层

### A 类：一票否决，必须拦截

出现任意一项，就不能直接进入 Stage 06：

1. 黑边 / 假 letterbox / 画中画边框
   - 上下黑条
   - 左右黑条
   - 内嵌相框式边框
   - 假胶片边框

2. 多图 / 宫格 / 拼贴
   - 一张输出里出现多个分镜格
   - 画面被切成多个面板

3. 主体错误
   - 多出第二个人
   - 主角脸明显换人
   - 发型明显换型
   - 裙子主结构明显变化

4. 构图不可用
   - 主体被严重裁断
   - 关键动作不可读
   - 完全看不清主角身份锚点

5. 明显生成事故
   - 多余肢体
   - 手部严重畸形且抢主视线
   - 身体结构断裂
   - 五官严重扭曲

### B 类：允许保留，但必须人工复核

这些情况不一定废片，但不能直接放行：

1. 与相邻镜头构图过近
   - 景别差异太小
   - 朝向差异太小
   - 人物占比几乎一样
   - 看起来像同一镜头重复抽卡

2. 情绪推进不明显
   - S001 和 S003 的情绪层级看不出差别
   - 画面没有体现 storyboard 里定义的推进点

3. 背影视角锚点变弱
   - 脸部信息减少后，只剩发型和裙型勉强识别
   - 虽然没换人，但身份可信度下降

4. 环境呼吸过强，吃掉主角
   - 海面和夕阳很好看，但人物在画面里不够可读

### C 类：可放行

满足下面条件，可以作为 Stage 06 候选帧：

1. 单张完整图像
2. 没有黑边、宫格、内嵌边框
3. 单主角稳定
4. 脸、发型、裙型主结构连续
5. 镜头意图清楚
6. 和相邻镜头存在可感知差异
7. 画面可直接作为 `single_subject_motion` 的首尾参考

## 第二轮 smoke 的最低质检清单

每张图至少检查：

1. 有没有黑边或内嵌边框
2. 有没有多人或副主体抢镜
3. 脸 / 发型 / 裙型是否稳定
4. 景别是否符合 prompt
5. 朝向是否符合 prompt
6. 人物在画面中的占比是否符合 prompt
7. 相邻镜头是不是明显不同

## 与 Stage 06 的衔接规则

### 可以进入 Stage 06

当输出满足：

- 单主角稳定
- 无黑边
- 无宫格
- 无明显身份漂移
- 构图适合作为起始或结束帧

### 不要直接进入 Stage 06

当输出出现：

- 黑边
- 近似重复镜头
- 只剩背影且身份锚点太弱
- 主体太小导致动作无法读清

这种情况应该：

1. 先补抽
2. 或改 prompt 收敛规则
3. 再决定是否送入 Stage 06

## 对代理的硬规则

1. 不要把“出图成功”汇报成“Stage 05 可交付”
2. 不要把“人物大体像”汇报成“人物一致性完全达标”
3. 背影视角类图必须额外检查黑边和身份锚点
4. 发现相邻镜头过近时，优先改 prompt，不要先盲目多抽
